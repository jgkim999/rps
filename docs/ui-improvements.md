# UI 디자인 개선

## 개요

Login과 Lobby 페이지에 Bootstrap CSS와 Bootstrap Icons를 적용하여 사용자 경험을 개선했습니다.

## 변경 사항

### 1. Bootstrap CSS 추가

**파일**: `Rps/Components/App.razor`

**변경 전**:
```html
<link rel="stylesheet" href="~/css/site.css" />
```

**변경 후**:
```html
<link rel="stylesheet" href="/lib/bootstrap/dist/css/bootstrap.min.css" />
<link rel="stylesheet" href="/css/site.css" />
```

**이유**:
- Bootstrap의 유틸리티 클래스(`fw-bold`, `mb-4`, `text-muted` 등) 사용 가능
- Bootstrap Icons (SVG) 사용 가능
- 반응형 그리드 시스템 활용
- 일관된 스타일링 제공

## 기술적 세부사항

### Bootstrap Icons 사용

SVG 인라인 방식으로 Bootstrap Icons를 사용했습니다:

```html
<svg xmlns="http://www.w3.org/2000/svg" width="20" height="20" fill="currentColor" class="bi bi-icon-name me-2" viewBox="0 0 16 16">
    <path d="..."/>
</svg>
```

**장점**:
- 외부 파일 로드 불필요 (번들 크기 최소화)
- `currentColor`로 상위 요소 색상 상속
- CSS로 크기와 색상 제어 가능
- 접근성 향상 (aria-hidden, visually-hidden 사용 가능)

### 반응형 디자인

기존 `site.css`의 반응형 미디어 쿼리 유지:
- `@media (max-width: 768px)`: 태블릿 최적화
- `@media (max-width: 576px)`: 모바일 최적화

### 색상 시스템

Bootstrap의 시맨틱 색상 클래스 활용:
- `text-success`: 승리 통계 (녹색)
- `text-danger`: 패배 통계 (빨간색)
- `text-secondary`: 무승부 통계 (회색)
- `text-muted`: 보조 텍스트 (회색)
- `text-primary`: 로딩 스피너 (파란색)

### 그라데이션 텍스트

CSS 그라데이션 클리핑으로 타이틀 강조:

```html
<h1 style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
           -webkit-background-clip: text;
           -webkit-text-fill-color: transparent;
           background-clip: text;">
    🎮 가위바위보
</h1>
```
